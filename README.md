# OPERATION TRACKER

A real-time intelligence dashboard for monitoring the Iran/Israel/US conflict.
Every data point is sourced from a verified, cited external source.
**No statistics are generated, estimated, or hallucinated by this application.**

## Getting API Keys (All Free)

### 1. ACLED — Required for Map & Strike Log

ACLED provides the gold-standard conflict event database with GPS coordinates and sourced fatality data.

1. Go to **https://developer.acleddata.com/**
2. Click "Register for Access"
3. Fill in your name, email, and intended use (e.g., "research / informational dashboard")
4. You will receive an email with your API key within 1-2 business days
5. Copy your key and email into `.env.local`:
   ```
   ACLED_KEY=your_key_here
   ACLED_EMAIL=your_email_here
   ```

### 2. EIA — Required for Oil Price Widget

The US Energy Information Administration offers free API access.

1. Go to **https://www.eia.gov/opendata/register.php**
2. Enter your email address
3. Your API key is shown **immediately** on the next page
4. Add to `.env.local`:
   ```
   EIA_KEY=your_key_here
   ```

### 3. NewsAPI — Optional

Used for the "Check for Updates" button in the Gov & Funding tab. Falls back to RSS feeds if not configured.

1. Go to **https://newsapi.org/register**
2. Create a free account (free tier: 100 requests/day)
3. Copy your API key to `.env.local`:
   ```
   NEWS_API_KEY=your_key_here
   ```

## Running Locally

```bash
# 1. Install dependencies
npm install

# 2. Set up environment variables
cp .env.local.example .env.local
# Edit .env.local and add your API keys

# 3. Start development server
npm run dev

# 4. Open browser to http://localhost:3000
```
